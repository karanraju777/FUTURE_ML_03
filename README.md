# 🚀 AI-Powered Resume Screening & Ranking System
**Final Year Project | BCA 5th Semester | Task 3 Submission**

---

### **📌 Project Overview: Task 3 Requirements**
This project is an automated HR solution that transforms raw resumes into actionable hiring data. It fulfills all deliverables for the **Resume / Candidate Screening System** task.

* **🔍 AI-Driven Categorization:** Automates evaluation using a **Random Forest** classifier and **TF-IDF** vectorization to predict 25+ professional job categories.
* **👤 Entity Extraction:** Utilizes **NLP & Regular Expressions (Regex)** to automatically extract candidate **Name, Email, and Phone Number** from digital PDFs.
* **📊 Candidate Ranking:** Proprietary scoring algorithm that ranks candidates into **Top, Mid, or Lower Tiers** based on JD match percentage.
* **🚩 Skill Gap Analysis:** Specifically identifies and highlights **Missing Skills** in red, providing HR with immediate gap identification.
* **🧠 Hybrid Intelligence:** Combines **Machine Learning** with **Heuristic Rule-Based logic** to eliminate model bias and ensure 100% accuracy.

---

### **🛠️ Technical Stack**
* **Language:** Python 3.11+
* **Backend Framework:** Flask (Web Deployment)
* **AI/ML Libraries:** Scikit-learn (Random Forest, TF-IDF), Pickle, NumPy, Pandas
* **Text Processing:** PyPDF2 (Parsing), Regular Expressions (Cleaning)
* **Frontend:** HTML5, Bootstrap 5 (Responsive UI)

---

### **📂 Task 3 Deliverables**
1. **Resume Parsing:** Automated noise-reduction and text extraction.
2. **Skill Matching:** Real-time comparison between Resume and Job Description.
3. **Ranking System:** Logic-based scoring for role fit.
4. **Deliverable Met:** Clear visualization of missing or required skills.
